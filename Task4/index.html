Projects

WEB DELOPMENT

The HTML defines a page with a title, heading ("Tic Tac Toe"), a restart button, and a gameboard containing individual squares. The CSS styles the page with an orange color scheme, a custom font, and positions the elements for a clean layout. The JavaScript manages the game's core functionality: It keeps track of the current player (X or O) and which boxes are filled with which symbol. When a box is clicked, the game checks if it's empty and fills it with the current player's symbol. The code also checks for winning combinations after each move. If a player wins, the winning squares are highlighted, and the winner is announced in the heading. The "Restart" button resets the game to its initial state.
